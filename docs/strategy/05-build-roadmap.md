# Suggested Build Roadmap

## Phase 1 — Fast MVP
Build first:
1. white-labeled product family pages
2. partner tracking URLs
3. one funding match quiz
4. one startup estimator
5. one working-capital page
6. CRM + webhook capture
7. simple attribution dashboard

## Phase 2 — Distribution System
Build next:
1. vertical pages
2. problem-solution pages
3. embeddable widgets
4. partner microsites
5. stronger email sequences
6. AI-assisted triage

## Phase 3 — Marketplace Layer
Build after:
1. searchable marketplace
2. recommendation engine
3. partner portal
4. internal product map UI
5. smarter routing rules
6. programmatic SEO system

## Phase 4 — Platform Layer
Eventually:
1. authenticated partner dashboard
2. analytics and attribution reporting
3. partner asset library
4. internal rule management
5. API-driven lead routing
6. AI agents for broker assist and triage

## What Not to Do

- do not publish a giant public spreadsheet
- do not let affiliates blast direct backend links
- do not expose provider names in slugs or metadata
- do not build tools without attribution and follow-up
- do not overbuild the CRM before validating the funnel
